Workation Backend

Product 엔드포인트 가이드

이 문서는 현재 com.kh.app.product 패키지의 컨트롤러 기준으로 정리한 API 목록입니다. 프론트 Axios 설정은 baseURL = http://localhost:80/api이므로 프론트에서는 /seller/space처럼 /api를 제외하고 호출합니다.

서버 기준 URL http://localhost:80/api
인증 방식 Authorization: Bearer {accessToken}
작성 기준 2026-05-29 현재 product 컨트롤러

공통 규칙

권한 prefix: /api/public/**는 비로그인 허용, /api/seller/**는 SELLER 또는 ADMIN, /api/admin/**은 ADMIN 권한이 필요합니다.
구분 규칙
소프트 삭제 DELETE는 hard delete가 아니라 엔티티의 delYn을 변경합니다.
노출 상태 visibleYn은 Y 또는 N입니다. Public 조회는 기본적으로 visibleYn=Y, delYn=N만 조회합니다.
셀러 소유권 Seller API는 JWT의 memberId로 본인 소유 Space, Stay만 접근합니다.
Multipart 등록 공간/스테이 등록은 multipart/form-data이며 JSON DTO는 dto, 이미지 파일은 files 파트로 보냅니다.

Public Space

Method Endpoint 권한 Query / Body 설명
GET /api/public/space PUBLIC keyword?, area? 공개 공간 목록 조회. visibleYn=Y, delYn=N 강제.
GET /api/public/space/{id} PUBLIC id 공개 공간 상세 조회. 공개되지 않았거나 삭제된 공간은 404.
GET /api/public/product/dummy-images/** PUBLIC 이미지 상대 경로 개발 더미 이미지 파일 응답. 예: /api/public/product/dummy-images/gangwon/hotel1/강원1외관.png

Seller Space

Method Endpoint 권한 Query / Body 설명
POST /api/seller/space SELLER multipart: dto, files? 공간 등록. 등록 셀러는 JWT에서 가져오며 DTO에 sellerId를 넣지 않습니다.
GET /api/seller/space SELLER 없음 본인 소유 공간 목록 조회. 삭제/비노출 포함.
GET /api/seller/space/{id} SELLER id 본인 소유 공간 상세 조회. 타 셀러 공간은 403.
PUT /api/seller/space/{id} SELLER SpaceUpdateReqDto 본인 소유 공간 기본 정보 수정.
DELETE /api/seller/space/{id} SELLER id 본인 소유 공간 삭제 처리.
PUT /api/seller/space/{id}/visible SELLER visibleYn=Y|N 본인 소유 공간 노출 여부 변경.
PUT /api/seller/space/{id}/pictures SELLER multipart: dto, files? 공간 사진 수정. keepPictureIds에 없는 기존 사진은 삭제하고 새 파일을 추가합니다.

Admin Space

Method Endpoint 권한 Query / Body 설명
GET /api/admin/space ADMIN keyword?, area?, visibleYn?, delYn? 전체 공간 목록 조회. 삭제/비노출 필터 선택 가능.
GET /api/admin/space/{id} ADMIN id 공간 상세 조회. 삭제된 공간 포함.
PUT /api/admin/space/{id} ADMIN SpaceUpdateReqDto 소유권 검증 없이 공간 수정.
DELETE /api/admin/space/{id} ADMIN id 공간 삭제 처리.
PUT /api/admin/space/{id}/visible ADMIN visibleYn=Y|N 공간 노출 여부 변경.

Public Stay

Method Endpoint 권한 Query / Body 설명
GET /api/public/stay PUBLIC keyword?, spaceId?, workationYn?, area?, minPrice?, maxPrice?, capacity?, options? 공개 스테이 목록 조회. visibleYn=Y, delYn=N 강제.
GET /api/public/stay/{id} PUBLIC id 공개 스테이 상세 조회. 공개되지 않았거나 삭제된 스테이는 404.

Seller Stay

셀러 스테이 목록은 반드시 /api/seller/stay를 사용해야 합니다. /api/public/stay를 호출하면 모든 공개 스테이가 조회됩니다.
Method Endpoint 권한 Query / Body 설명
POST /api/seller/stay SELLER multipart: dto, files? 스테이 등록. spaceId가 본인 소유 공간이어야 합니다.
GET /api/seller/stay SELLER 없음 본인 소유 공간에 속한 전체 스테이 조회. 삭제/비노출 포함.
GET /api/seller/stay/{id} SELLER id 본인 소유 스테이 상세 조회. 타 셀러 스테이는 403.
PUT /api/seller/stay/{id} SELLER StayUpdateReqDto 본인 소유 스테이 정보 수정.
PUT /api/seller/stay/{id}/visible SELLER visibleYn=Y|N 본인 소유 스테이 노출 여부 변경.
DELETE /api/seller/stay/{id} SELLER id 본인 소유 스테이 삭제 처리.

Admin Stay

Method Endpoint 권한 Query / Body 설명
GET /api/admin/stay ADMIN keyword?, spaceId?, workationYn?, area?, minPrice?, maxPrice?, capacity?, options?, visibleYn?, delYn? 전체 스테이 목록 조회. 삭제/비노출 필터 선택 가능.
GET /api/admin/stay/{id} ADMIN id 스테이 상세 조회. 삭제된 스테이 포함.
POST /api/admin/stay ADMIN multipart: dto, files? 소유권 검증 없이 스테이 등록.
PUT /api/admin/stay/{id} ADMIN StayUpdateReqDto 소유권 검증 없이 스테이 수정.
DELETE /api/admin/stay/{id} ADMIN id 스테이 삭제 처리.
PUT /api/admin/stay/{id}/visible ADMIN visibleYn=Y|N 스테이 노출 여부 변경.

Arcade

Method Endpoint 권한 Query / Body 설명
GET /api/seller/arcade SELLER 없음 편의시설 전체 목록 조회.
